## DeepTutor 是什么

DeepTutor 是开源、agent-native 的个性化学习伴侣。共享 runtime 驱动 Chat、深层 capabilities、Partners、Book generation、Co-Writer editing、RAG 和三层 Memory。Partners 把同一个 chat agent loop 带到 IM 渠道里，同时拥有自己的 soul、library、tools 和 workspace。

由香港大学数据智能实验室构建。Apache 2.0 协议。
